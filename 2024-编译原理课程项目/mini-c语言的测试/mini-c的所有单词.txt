1.Mini C的单词

1).下面是语言的关键字:
e1se if  int  float double return void   while do
所有的关键字都是保留字,并且必须是小写。

2).下面是专用符号:

_  + - * / % ^ <  <= > >= ==  !=  =  ;  , ( ) [ ] { }  //行注解

3).其他标记是ID和NUM ，通过下列正则表达式定义:
ID= (_|letter)(_|letter|digit)*
NUM=digit+(.digit+)?
letter= [a-zA-Z]
digit=[0-9]

小写和大写字母是有区别的。


4).空格由空白、换行符和制表符组成。空格通常被忽略,除了它必须分开ID、NUM关键字。

5).注释用通常的C语言符号//是行注解且不可以超过一行。注释不能嵌套。
